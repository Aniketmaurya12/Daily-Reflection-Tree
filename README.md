Daily Reflection AI System

This project implements a deterministic decision tree and an AI agent that helps users reflect on their daily productivity.

Part A:
A deterministic decision tree that evaluates daily performance based on task completion, satisfaction, and distractions.

Part B:
An AI agent that:
- Validates input
- Uses deterministic reasoning
- Logs decisions
- Prevents hallucination

Guardrails:
1. Input validation
2. Deterministic rules
3. Logging
4. No random outputs

How to Run:

python decision_tree.py

python agent.py
